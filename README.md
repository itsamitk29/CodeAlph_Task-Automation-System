# 🚀 Task Automation System

A simple Python project that automates common tasks using Object-Oriented Programming (OOP). This project provides a menu-driven interface where users can choose different automation tasks.

## 📌 Features

- 📂 Move JPG files from one folder to another
- 🖼️ Move PNG files from one folder to another
- 📧 Extract email addresses from a text file
- 🌐 Scrape the title of a webpage and save it into a file
- 📋 Menu-driven interface
- 🛡️ Basic error handling

## 🛠️ Technologies Used

- Python 3
- Object-Oriented Programming (OOP)
- os
- shutil
- re (Regular Expressions)
- requests

## ▶️ How to Run

### 1. Clone the repository

```bash
git clone https://github.com/your-username/Task-Automation-System.git
```

### 2. Open the project folder

```bash
cd Task-Automation-System
```

### 3. Install the required package

```bash
pip install requests
```

### 4. Run the project

```bash
python automation.py
```
